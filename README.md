\*\*Highlights\*\*

\- Automated metadata discovery \& PII classification.  

\- Business Glossary creation \& linkage to datasets.  

\- Stewardship assignment and compliance metrics via Purview Insights.  



\*\*Tools:\*\* Microsoft Purview | Azure Blob Storage | Azure Portal | Power BI.
